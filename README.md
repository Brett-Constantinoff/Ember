# Ember Engine
 The Ember engine is a work in progress cross platform 3D rendering engine using the OpenGL sepcification. 

 ## Build
 ### Windows
 To build run ```bin\premake\windows\premake5 vs2022```
 
 ### MacOS and Linux
 To build run ```bin/premake/macos/premake5 gmake```
